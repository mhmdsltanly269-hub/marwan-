# 🎓 مروان أكاديمي - Mervan Academic

تطبيق ويب احترافي لإعادة صياغة النصوص المولدة بالذكاء الاصطناعي إلى صياغة بشرية أكاديمية.

## ✨ المميزات
- 🔐 نظام تسجيل دخول احترافي
- 📝 إعادة صياغة حتى 20,000 كلمة
- 📁 رفع ملفات PDF, Word, TXT
- 📥 تحميل النتيجة بصيغة Word
- 🎨 تصميم زجاجي فاخر بتموجات ذهبية
- 📱 متجاوب مع جميع الأجهزة

## 🚀 التشغيل

```bash
git clone https://github.com/mervan/academic.git
cd mervan-academic
pip install -r requirements.txt
python app.py
